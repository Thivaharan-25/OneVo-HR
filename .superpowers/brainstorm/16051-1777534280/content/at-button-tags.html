How it works

  • Developer clicks the @ button in the composer
  • Picker popup appears — searchable, grouped by category
  • HR section only shows if the tenant has HR module enabled
  • Each HR tag is filtered by what the user is allowed to do (permissions check)
  • Click @clockin → inserts into message → sends → bot confirmation card appears → confirm → action fires
  • Works identically to typing @clockin manually in chat
