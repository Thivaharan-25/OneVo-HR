# Page: Exception Management

**Route:** `/workforce/exceptions`
**Permission:** `exceptions:view`
**Real-time:** SignalR `exception-alerts` channel

## Purpose

View, acknowledge, dismiss, and manage exception alerts generated by the exception engine.

## Actions

| Action | Permission | API |
|:-------|:-----------|:----|
| Acknowledge | `exceptions:acknowledge` | `PUT /exceptions/alerts/{id}/acknowledge` |
| Dismiss | `exceptions:acknowledge` | `PUT /exceptions/alerts/{id}/dismiss` |
| Configure rules | `exceptions:manage` | Navigate to `/settings/monitoring` |

## Real-time

- New alerts appear at the top of the list via SignalR
- Toast notification on new critical/warning alerts
- Badge count in sidebar navigation updates
