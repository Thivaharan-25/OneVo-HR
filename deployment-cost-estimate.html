Cloud Deployment Cost Estimate

Infrastructure sizing and monthly operating cost for the OneVo Platform — Clean Architecture edition (Phase 1 · Single Region · Azure + Cloudflare)

Executive Summary
What this document covers This estimate covers the recurring cloud infrastructure cost to deploy and operate the OneVo Platform (HR Management + Workforce Intelligence). The stack combines Microsoft Azure for compute, database, and caching with Cloudflare for global delivery, WAF security, and zero-egress object storage — delivering a production-grade SaaS setup at minimal cost. All prices are in USD based on Southeast Asia (Singapore) region pricing as of Q1 2026.

Architecture Basis
Deployment Model Single deployable unit (Modular Monolith). All 23 backend modules ship as one ASP.NET Core host. Inter-module communication uses in-process MediatR events — no external message broker required. This reduces infrastructure complexity and lowers baseline cost significantly.

Assumptions & Notes
  • Azure prices are based on Southeast Asia (Singapore) region · 1-Year Reserved pricing as of Q1 2026. Cloudflare prices are global and region-independent.
  • The platform runs as a single-region, single-instance deployment in Phase 1. No horizontal scaling is assumed at this cost level.
  • SignalR is handled in-process within the single App Service instance at no extra cost. Azure SignalR Service (~$49/month) is only required when scaling to 2+ instances.
  • Cloudflare R2 replaces Azure Blob Storage, eliminating egress fees on document and payslip downloads. Storage cost grows at $0.015/GB beyond the 10 GB free tier.
  • Cloudflare DNS/WAF/CDN replaces Azure Front Door, saving approximately $35/month. Upgrade to Cloudflare Pro ($20/month) when custom WAF rules are required.
  • The WorkPulse Agent (Windows desktop) processes activity data client-side. Only lightweight JSON payloads are sent to the API — no additional server compute cost.
  • Database backup costs are included within the PostgreSQL Reserved pricing tier (7-day automated backup included by default).
  • Outbound bandwidth estimate assumes Cloudflare proxying reduces origin requests by ~70%; actual cost depends on screenshot capture frequency.
  • Costs do not include Azure Support Plan. Recommended: Developer tier at $29/month for a production workload.
